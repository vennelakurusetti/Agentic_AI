# AI Quiz Generator

## Goal
Upload PPT and generate MCQ quiz using AI.

## Input
- PPT/PPTX file
- Question count (5-30)
- Difficulty (Simple/Medium/Complex)

## Output
- Interactive quiz
- Score
- Correct answers
- AI explanations

## Flow
1. Upload PPT
2. Extract slide text
3. Generate MCQs
4. Take quiz
5. Show results

## Done When
- PPT upload works
- Quiz generated
- Score displayed
- Feedback shown
