# Agentic_AI

This repository contains a collection of mini-projects focused on agentic AI, prompt engineering, and interactive AI applications.

## Projects

- AIQuizGenerator: A small app for generating quizzes with AI assistance.
- multimodel-app: A multi-model application demo that explores model interactions.
- prompt_doctor: A prompt evaluation and improvement project.
- prompt_pipeline: A simple prompt pipeline example and interface.

## Purpose

These projects are intended for learning, experimentation, and demonstrating practical use cases of AI agents and prompt-driven workflows.

## Getting Started

1. Clone the repository.
2. Open any project folder and follow its local README or setup instructions.
3. Install dependencies using the relevant requirements file if present.

## Technologies

- Python
- Streamlit / web-based interfaces
- Prompt engineering and AI application experiments
